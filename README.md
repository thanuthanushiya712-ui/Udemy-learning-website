# Udemy-learning-website
Udemy Clone Learning Website

A responsive Udemy-inspired learning platform built using HTML5 and CSS3. This project recreates the basic user interface of an online learning website, featuring a modern navigation bar, course categories, recommended courses, and a clean, user-friendly design. It demonstrates fundamental web development skills, responsive layouts, and structured webpage design without using JavaScript.

Key Features:

Responsive and modern UI
Navigation bar with search functionality
Course recommendation section
Mobile-friendly layout
Built entirely with HTML and CSS

Technologies Used:

HTML5
CSS3

This project was created to strengthen my front-end web development skills and gain hands-on experience in designing real-world website layouts. 🚀
